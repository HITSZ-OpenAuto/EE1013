包含原《电路IA》前四章、原《模拟电子技术基础》和原《数字电子技术基础》前若干章。

graph LR

B(数电1-2章)

C-->D
B-->E

上图中，方框中为选学内容（在预习的有限时间里优先级较低），圆框中为必学内容；标有双箭头的框之间可同步学习，标有单箭头的框之间有承接关系。
